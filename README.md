# Prediction of Bottles Image

## Bakcground 
This program was created to detect bottle images, the images will be classified into several categories such as Beer bottles, Plastic bottles, Soda bottles, Water bottles and Wine bottles. 

## Tools 
Glob, Numpy, TensorFlow, Matplotlib, Keras.
